Machine Learning Model for Prediction of Smartphone Addiction

Understanding Smartphone Addiction

Smartphone addiction is a growing concern in today's digital age, affecting individuals' mental health, productivity, and social interactions. It is characterized by excessive use of smartphones, leading to negative impacts on daily life. Understanding the factors contributing to smartphone addiction is essential for developing effective interventions and promoting healthier usage patterns.

Data Collection and Analysis

This project begins with collecting a comprehensive dataset that includes various metrics related to smartphone usage, such as screen time, frequency of app usage, and user engagement levels. By analyzing this data using machine learning algorithms, we can identify patterns that signify potential addiction. Feature selection and engineering play a crucial role in enhancing the accuracy of our predictive model.

Model Development

We employ various machine learning techniques, including Random Forest, Logistic Regression, and Support Vector Machines, to develop our prediction model. The effectiveness of each model is evaluated using metrics such as accuracy, precision, recall, and F1-score. The Random Forest algorithm, in particular, excels in handling complex datasets and provides insights into the importance of different usage features influencing smartphone addiction.

Implementation and Impact

The final model is deployed within a user-friendly web application that allows users to input their smartphone usage data and receive real-time predictions regarding their addiction levels. By utilizing these predictions, individuals and organizations can make informed decisions to promote healthier smartphone usage habits, ultimately leading to improved well-being and productivity.
